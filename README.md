# Python Slot Machine

A simple command-line slot machine game built using Python while learning programming fundamentals.

## Features

* Place bets and spin the slot machine
* Random symbol generation
* Win rewards for matching symbols
* Balance tracking system

## Technologies Used

* Python
* Random module

## What I Learned

* Variables and data types
* Conditional statements
* Loops
* Functions
* User input handling

## Future Improvements

* Leaderboard system
* Save game progress
* More symbols and rewards
